엔트리-Aanmeldkontrole 개인정보처리방침

최종 수정일: 2025년 11월 13일

1. 개요
엔트리-Aanmeldkontrole(이하 "본 확장 프로그램")은 사용자의 개인정보 보호를 최우선으로 합니다. 본 개인정보처리방침은 본 확장 프로그램이 어떠한 정보를 수집하고 사용하는지 설명합니다.

2. 수집하는 정보
본 확장 프로그램은 어떠한 개인정보도 수집하지 않습니다.
- 사용자의 이름, 이메일, 연락처 등 개인 식별 정보를 수집하지 않습니다
- 브라우징 기록을 수집하거나 저장하지 않습니다
- 로그인 정보(아이디, 비밀번호)를 수집하거나 저장하지 않습니다
- 쿠키나 기타 추적 기술을 사용하지 않습니다

3. 확장 프로그램의 작동 방식
본 확장 프로그램은 다음과 같이 작동합니다:
- 사용자가 방문하는 웹사이트의 도메인이 playentry.org인지 확인합니다
- 공식 사이트가 아닌 경우, 페이지 구조를 분석하여 엔트리 로그인 페이지를 모방하는지 확인합니다
- 피싱 사이트로 판단되는 경우, 로그인 버튼을 비활성화하고 경고 메시지를 표시합니다
- 모든 처리는 사용자의 브라우저 내에서만 이루어지며, 외부 서버로 데이터를 전송하지 않습니다

4. 데이터 저장 및 전송
본 확장 프로그램은:
- 어떠한 데이터도 로컬 또는 원격 서버에 저장하지 않습니다
- 제3자에게 정보를 공유하거나 전송하지 않습니다
- 분석 도구나 광고 서비스를 사용하지 않습니다

5. 권한 사용
본 확장 프로그램이 요청하는 권한과 그 사용 목적:
- 모든 웹사이트 접근 권한(<all_urls>): 피싱 사이트는 어떤 도메인에서도 나타날 수 있으므로, 사용자를 보호하기 위해 모든 웹사이트에서 공식 사이트 여부를 확인해야 합니다. 이 권한은 오직 도메인 확인 및 로그인 버튼 비활성화 목적으로만 사용되며, 어떠한 데이터도 수집하지 않습니다.

6. 보안
본 확장 프로그램은 사용자의 보안을 강화하기 위해 설계되었으며:
- 외부 네트워크 통신을 하지 않습니다
- 소스 코드는 공개되어 있어 투명성을 보장합니다
- 사용자의 정보에 접근하거나 수정하지 않습니다

7. 어린이 개인정보 보호
본 확장 프로그램은 어떠한 연령대의 사용자로부터도 개인정보를 수집하지 않습니다.

8. 개인정보처리방침 변경
본 개인정보처리방침은 필요에 따라 업데이트될 수 있습니다. 중요한 변경사항이 있을 경우, 확장 프로그램 업데이트를 통해 사용자에게 알립니다.

9. 문의
본 개인정보처리방침에 대한 질문이나 우려사항이 있으시면 Chrome Web Store의 본 확장 프로그램 페이지를 통해 연락주시기 바랍니다.

요약:
본 확장 프로그램은 사용자의 개인정보를 전혀 수집, 저장, 전송하지 않습니다. 모든 기능은 사용자의 브라우저 내에서만 작동하며, 오직 피싱 사이트로부터 사용자를 보호하는 목적으로만 사용됩니다.
